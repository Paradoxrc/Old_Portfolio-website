# My Portfolio

Welcome to my portfolio! This project showcases my skills, projects, and experiences as a web developer.

- [Features](#features)

## Introduction
This portfolio is built using React.js to showcase my work, skills, and achievements. It provides an overview of my background, features my notable projects, and offers a way to contact me.

## Features
- **About Me**: Overview of my background, skills, and experiences.
- **Projects**: A collection of my projects with descriptions, screenshots, and links.
- **Skills**: A list of my technical skills.
- **Blog**: A section with links to my articles.
- **Resume**: A downloadable version of my resume.
- **Contact**: A form to send inquiries or messages.

## Technologies Used
- **React.js**: JavaScript library for building user interfaces.
- **HTML5 & CSS3**: For structuring and styling the web pages.
- **Styled-Components**: For styling React components with CSS.
- **JavaScript**: For adding interactivity to the web pages.
- **Git**: For version control.
- **Netlify**: For hosting and deployment.
